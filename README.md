##  Project Overview

The Resume Builder Web Application is a full-stack web platform designed to simplify the process of creating professional resumes. It provides a structured and user-friendly interface where users can input their  personal details, education, skills, and experience, and generate a well-structured resume instantly.

---

##  Features
-  User Authentication (Login / Signup)
-  Multiple Resume Sections (Education, Skills, Experience)
-  Live Resume Preview
-  Download Resume in PDF format
-  Clean and User-Friendly Interface
-  Fully Responsive Design
